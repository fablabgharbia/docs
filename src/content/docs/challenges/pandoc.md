# pandoc

## what is pandoc

`If you need to convert files from one markup format into another, pandoc is your swiss-army knife`

## how to install

- `sudo apt-get update`
- `sudo apt-get install pandoc`
- `apt install dpkg`
- `dpkg -L pandoc`
- `pandoc -v`
